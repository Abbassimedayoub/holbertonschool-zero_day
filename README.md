# Zero Day — First Repository

## Overview

My very first Holberton School repository: discovering **Git and the shell**. The `git/` directory contains the exercises of the initial Git project (cloning, committing, branching, merging, pushing).

> Note: the commit messages in this repository ("My first commit", "Never push too much"…) are part of the exercise itself and are kept as-is — they document the very beginning of the journey.

## Skills Demonstrated

Git fundamentals: repository setup, staging, commits, branches, merges and remotes.
